# Proyecto Intermodular
Cuando hagas cambios **git add .   git commit -m "mensaje explicando el commit"  git push**
Lo que te falta por hacer: 
-Crear la Base de datos en donde quieras: xampp o heidy
-Fronted de la app web, HTML y CSS
-Conectar la base de datos con la app (servidor API REST)
-Diseño movil
-Usar PHP (Laravel) o lo que veas conveniente para poder conectar la pagina web con la base de datos y darle un diseño con programacion más extansa que con JS, HTML o CSS
